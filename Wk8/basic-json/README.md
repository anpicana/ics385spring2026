# University Course Catalog

Description:

This project is a University Course Catalog application that allows users to view, add, and manage course data stored in JSON format. 
It includes features for data validation and error handling to ensure data consistency. 
This project is useful for students, faculty, and developers who want a simple way to organize course information.

## Features
- View all courses with details (course code, title, description)
- Add new courses with validation
- Data validation and error handling for JSON files
- Optional browser interface to display courses
