HOW TO PLAY SIMON


1. Press any key to start the game

2. Watch the color sequence - The game will flash a series of colored buttons

3. Repeat the sequence - Click the same buttons in the same order

4. Each round gets harder - One more color is added to the sequence

5. Don't make mistakes! - If you click the wrong button, the game ends

6. How far can you go? - See if you can remember longer and longer sequences!
